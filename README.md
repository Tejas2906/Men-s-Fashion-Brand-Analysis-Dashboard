# Men’s Fashion Brand Analysis Dashboard | Power BI

An interactive **Men’s Fashion Brand Analysis Dashboard** developed using **Power BI** to evaluate brand performance, pricing, discounts, and profitability.  
This project was created as part of my **learning and hands-on practice with Power BI**, focusing on data visualization, DAX measures, and dashboard storytelling.

---

## Project Purpose

The main objectives of this project are to:
- Analyze **brand-level performance** in men’s fashion
- Identify brands offering **higher discounts**
- Compare **average sales price and profit**
- Understand **variety availability** across brands
- Practice designing a **business-focused Power BI dashboard**

---

## Tools & Technologies Used

- **Power BI**

- **Power Query**

---

## Dataset Information

- Source: Men’s T-shirt sales dataset (CSV format)
- Includes:
  - Brand names
  - Discount percentage
  - Sales price and profit
  - Product variety count
- Data Preparation:
  - Data cleaning and transformation
  - Creation of calculated measures using DAX
  - Structuring data for analytical reporting
